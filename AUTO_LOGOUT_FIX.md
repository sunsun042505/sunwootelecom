# 로그인 후 자동 로그아웃 수정

원인:
로그인 시 세션 토큰을 Netlify Blobs에 저장하고,
대시보드 진입 시 `/me`가 다시 Blobs에서 세션을 찾는 구조였음.
Netlify Blobs 저장 반영 타이밍이나 읽기 지연이 있으면 로그인 직후 `/me`에서 세션을 못 찾아 자동 로그아웃처럼 보일 수 있음.

수정:
세션을 Blobs에 저장하지 않고, 서명된 토큰 안에 branchId/employeeId를 담도록 변경함.

변경 후:
- 로그인 성공 → 토큰 발급
- 대시보드 `/me` → 토큰 서명 검증 → 직원/지점 확인
- Blobs 세션 저장 지연 문제 없음

선택 설정:
Netlify 환경변수에 `NEVERLAB_TOKEN_SECRET`을 추가하면 토큰 서명 키를 직접 지정할 수 있음.
없어도 기본값으로 동작함.
